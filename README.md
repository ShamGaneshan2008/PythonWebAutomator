# Python Web Automation (Selenium)

A Python-based browser automation project built using Selenium WebDriver.
This project automates common web tasks such as logging into a website, filling out forms, and downloading files.

## 🚀 Features

* Automated browser launch using Selenium
* Login automation for websites
* Automatic form filling
* File download automation
* Structured and modular Python code
* ChromeDriver handled automatically using WebDriver Manager

## 🛠️ Technologies Used

* Python
* Selenium WebDriver
* WebDriver Manager
* Python Dotenv
* Chrome Browser Automation

## 📂 Project Structure

```
PythonWebAutomation
```

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/ShamGaneshan2008/PythonWebAutomation.git
```

Move into the project folder:

```
cd PythonWebAutomation
```

Create a virtual environment (optional but recommended):

```
python -m venv .venv
```

Activate the environment:

Windows:

```
.venv\Scripts\activate
```

Install dependencies:

```
pip install -r requirements.txt
```

## ▶️ Running the Project

Run the automation script:

```
python main.py
```

Or launch the GUI version:

```
python GUI.py
```
